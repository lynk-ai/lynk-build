# Isolation and strict briefs

**What it is** — the ISOLATE lever done properly. A subagent gets a fresh context window, does heavy work there, and returns only a condensed answer — its mess never touches the parent. But isolation creates a new problem: isolated workers are *blind*, so what you hand them decides everything.

**Takeaway** — **isolation does the containment, compression decides what escapes it, and the brief keeps blind workers from colliding.**
