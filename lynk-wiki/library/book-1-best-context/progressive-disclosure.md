# Progressive disclosure

> Superseded 2026-07-07: graduated to its own book (book-3-progressive-disclosure). Definition + takeaway kept here; full treatment moved.

**What it is** — loading context lazily: a small index up front, the full content only when a task actually needs it. The point is the economics — cost scales with what's *used*, not what *exists*, so a library can grow without making every session more expensive.

**Takeaway** — **pay a pointer for everything that exists; pay full price only for what the task actually touches.**
